# Seller Notes

This is a reusable agent workflow template, not a finished project config.
The value is in the structure and the discipline it gives the AI agent.

## What To Sell

Position it as:

- "AI coding agent operating system for any repo"
- "Project memory, rules, workflows and safety rails for Codex/GPT coding agents"
- "Drop-in `.agent` folder plus `AGENTS.md` and Custom Instructions"

## What To Promise

Safe promises:

- helps agents start with the right files;
- reduces context waste;
- protects user changes from accidental revert;
- creates searchable project memory;
- standardizes tickets, bug logs and deploy checks;
- makes commit/push scope clearer.

Avoid promising:

- bug-free code;
- automatic production safety;
- no need for human review;
- universal DB/deploy support without configuration.

## Client Setup Checklist

- [ ] Replace placeholders.

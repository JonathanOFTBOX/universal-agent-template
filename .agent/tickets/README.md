# Tickets

Put active tickets in this folder.

Recommended naming:

```text
TICKET-001-short-title.md
TICKET-002-short-title.md
```

Use `.agent/templates/ticket-template.md`.

Rules:

- If the user mentions a ticket, read it before editing code.
- Keep tickets scoped.
- Archive old tickets in `.agent/tickets/archive/` if desired.
- Do not store secrets or credentials in tickets.
